# Design System

## Fonts
- **Primary Font:** Times New Roman, Size 16pt
- **Secondary Font:** Times New Roman, size 13pt
## Colours
- **Primary Colour:** #F0E68C (Khaki)
- **Secondary Colour:** #006400 (Dark green)
- **Table colour:** #FFFFFF (White)
